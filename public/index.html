IMDb Lookup Endpoint

GET https://fmdbapi.vercel.app/?i={imdb_id}&apikey={your_api_key}

Parameters

i (required)

Valid IMDb ID (e.g. tt0111161)

apikey (required)

Your registered API key

Example Request

                  
curl "https://fmdbapi.vercel.app/?i=tt0111161&apikey={your_api_key}"

Example Response

                  
{
  "Title": "The Shawshank Redemption",
  "Year": "1994",
  "Type": "movie",
  "Poster": "https://m.media-amazon.com/images/M/...jpg",
  "Response": "True"
}

Title Search Endpoint

GET https://fmdbapi.vercel.app/?s={search_term}&page={page}&apikey={your_api_key}

Parameters

s (required)

Search term (e.g. "batman")

page (optional)

Page number (default: 1)

apikey (required)

Your registered API key

Example Request

                  
curl "https://fmdbapi.vercel.app/?s=batman&page=2&apikey={your_api_key}"

Example Response

                  
{
  "Search": [
    {
      "Title": "Batman Begins",
      "Year": "2005",
      "Type": "movie",
      "Poster": "https://m.media-amazon.com/images/M/...jpg"
    },
    ...
  ],
  "totalResults": "127",
  "Response": "True"
}
